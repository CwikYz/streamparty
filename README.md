# Sync video playback between browsers

Supported:
- Youtube

Soon to be supported:
- Vimeo
- Daily Motion

Finished:
- Sync play/pause/playback rate change/seek
- Sync new user on join
- Add videos to queue
- Play next/prev video
- Search YT videos
